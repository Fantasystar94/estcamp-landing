AI 휴먼,
어디에 사용할까?

AI 휴먼 제작 기술을 배우면 좋은 점이 무엇일까요?

🎪

전시회·박람회

24시간 전시 부스 안내부터 제품 소개, 데모 설명까지 자연스럽게 수행하며 관람객 경험을 향상합니다.

✈️

공항·쇼핑몰

전 세계 방문객을 다국어로 자연스럽게 안내합니다.

🏨

호텔·관광지

객실 예약, 체크인 안내부터 주변 맛집·관광지 추천까지 여행자가 필요한 정보를 즉시 제공하는 디지털 컨시어지입니다.

🛍️

쇼핑·커머스

다양한 외형의 글로벌 AI 휴먼들이 비즈니스 정장부터 캐주얼, 전통 의상까지 다양한 착장으로 원하는 분위기에 맞는 연출이 가능합니다.

🏟️

스포츠·페스티벌

팀·선수 정보, 경기기록, 현장 이벤트까지 실시간으로 알려주는 인터랙티브 응원 파트너입니다.

📚

도서관·공공기관

도서 검색, 시설 이용 안내, 민원 절차 등 복잡한 정보를 대화로 쉽게 이해시키는 스마트 큐레이터입니다.

당신의 첫 AI 휴먼 탄생 과정

01
Python 프로그래밍
Python 데이터 전처리 시각화
⌄
  • 상수, 변수, 함수, 제어문, 반복문을 이용한 코드 구현

  • Pandas, Numpy를 이용한 데이터 전처리
  • Matplotlib, Seaborn 등 도구를 활용한 데이터 시각화
02
머신러닝과 딥러닝
머신러닝 딥러닝 RNN/CNN/LSTM

  • 간단한 머신러닝 모델 학습을 통해 모델, 학습, 추론 등 개념 이해
  • 인공신경망과 기본적인 딥러닝 모델(RNN, CNN, LSTM 등) 학습
  • 딥러닝 모델을 사용한 문제 해결 실습
03
자연어 및 음성 데이터 활용 및 모델 개발
TTS/STT 음성 모델 멀티모달
⌄
  • 자연어 및 음성 데이터의 특성 파악 및 전처리 방식 학습
  • 분석을 위한 기초 모델에 대한 이해 및 멀티 모달 활용
  • 데이터의 특성에 따른 적절한 음성 모델 검증하기
  • 프로젝트음성 데이터 TTS와 STT 음성 모델 개발
04
거대 언어 모델을 활용한 자연어 생성
LLaMA Transformer 프롬프트 엔지니어링
⌄
  • LLaMA를 활용한 자연어 생성 실습
  • Transformer 신경망 아키텍쳐 모델 학습
  • 프롬프트 엔지니어링을 통한 답변 형식 최적화
  • 프로젝트거대 언어 모델을 활용한 음성 모델 답변 기능 구현하기
05
Langchain 활용 및 RAG 구현 + 파이널 프로젝트
Langchain RAG 나만의 AI 휴먼 완성
⌄
  • Langchain 프레임워크 활용법 이해 및 데이터베이스 통합 시스템 구축
  • RAG(Retrieval-Augmented Generation) 이해 및 외부 데이터 소스 연동
  • 파이널 프로젝트Langchain 활용 LLM 기반 AI 시스템 완성
  • 사용자 질의에 대해 정확하고 풍부한 맥락을 제공하는 커스텀 AI 시스템 완성
